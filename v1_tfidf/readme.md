# SentivaAI v1

SentivaAI is an AI-powered sentiment analysis application designed to classify Roman Urdu customer reviews into Positive, Neutral, and Negative sentiments using Machine Learning and Natural Language Processing.

## Features

- Roman Urdu text preprocessing
- Sentiment prediction
- Simple Streamlit interface
- Confidence score
- History tracking

## Tech Stack

- Python
- Streamlit
- Transformers
- PyTorch
- Pandas

## Project Structure

```
SentivaAI/
│
├── app.py
├── predictor.py
├── preprocessing.py
├── config.py
├── utils.py
├── requirements.txt
└── README.md
```

## Installation

```bash
git clone <repository-url>

cd SentivaAI

pip install -r requirements.txt

streamlit run app.py
```

## Output

- Positive Sentiment
- Neutral Sentiment
- Negative Sentiment
- Confidence Score
